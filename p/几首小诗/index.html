# 我的麦子黄了

我的麦子黄了
杏子也是
麻雀第一个发现
蹦跳着飞入夏天
梧桐叶落下
磕在了井台上
纸鹤在窗边落了灰
泛了很久的黄
铁犁生了锈
它在怀念有一年春天
刚苏醒的土地
有草根细细的崩断声
孩子们在等大渠开闸
像蒲公英一样聚集在河岸边
等着看水的老汉掰动黑黑的铁把手
开出一朵朵浪花
一截烟头
把野花花瓣烫出了一个圆
云被晒得跑到了天边翻涌
农人就开心了
傍晚起风的时候 要扬麦
用木锨把夏天挑起
在晚风里变凉
只剩下胖胖的麦粒
五粒麦子就可以在手臂上压一朵花
你有玩过吗
孩子们在麦秸里钻来钻去
鸡也得赶紧挑一个好地方下蛋
银河在头顶溅开变成星子
谁的口水流了半张脸
夜里九点
乡下就已经入眠了
只剩下莫名的野风
汹涌着即将收割的麦田
麦浪滚滚
明天又是新的一天
流萤纷飞四起
这是我们的夏天

# 小花

我和一排排花儿挤在一起晒太阳
马蹄踩过，带走花香
你看不见我，在原地受着伤
你回头了吗
不用你回头
我不愿你看到我这样
一朵花
注定要在原地发芽、长大、枯萎 陪月色一起冰冰凉
可它偏偏想陪你去流浪
你回头了吗
你离开得那么快
像风一样

# 等

风是好看的

我们一起站在山坡上开花

等风来，把我们吹得清凉又好看

月是微甜的

我们一起站在树下听雪

等着一个下弦月

淋成一树白山茶
